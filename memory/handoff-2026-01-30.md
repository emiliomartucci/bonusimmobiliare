# Handoff 2026-01-30

## Summary
Creata mappa interattiva zone OMI Roma con Leaflet. 233 zone con valori canone concordato, ricerca indirizzo con Nominatim, info panel con fasce canone. Template riutilizzabile per future citta.

## Cosa e cambiato
- `src/mappe/roma/index.html`: Mappa interattiva Leaflet (1200 righe)
- `src/data/roma_omi_zones.geojson`: Poligoni zone OMI ottimizzati (1.2MB da 2MB originali)
- `src/data/roma-zone-omi.json`: Valori canone per zona (235 zone, 3 fasce A/B/C)
- `src/index.html`: Aggiunto dropdown "Mappe" in navbar (desktop + mobile)
- `context.md`: Aggiunto sezione mappe, aggiornato struttura cartelle, versione 2.1.0

## Decisioni Prese
- **CITY_CONFIG object:** Variabili configurazione in testa al file per riutilizzabilita template su altre citta
- **Leaflet + Nominatim:** Librerie open source senza costi/limiti API significativi
- **GeoJSON ottimizzato:** Rimossi campi inutili (description HTML, tessellate, ecc.), precisione coordinate ridotta a 5 decimali
- **Design system inline:** CSS consistente con resto del sito (warm editorial aesthetic, colori olive/terracotta/cream)
- **CartoDB Positron tiles:** Stile mappa pulito ed editoriale, coerente con brand

## Discussioni/Reasoning
Dati zone OMI estratti da PDF Accordo Territoriale Roma 7 agosto 2023 (235 zone totali).
Zone rurali (R1, R2, ecc.) hanno valori null con nota "applicare valori zona confinante" come da Art. 2 accordo.
KML originale scaricato da GeoPOI Agenzia Entrate, convertito con ogr2ogr, poi ottimizzato con Python script.

## Come testare
1. Aprire http://localhost:8080/mappe/roma/ (o servire con live-server)
2. Verificare mappa carica con zone colorate per tipologia (B rosso, C arancio, D giallo, E verde, R oliva)
3. Cercare indirizzo (es. "Via del Corso, Roma")
4. Verificare popup mostra nome zona e range canone
5. Verificare info panel destro con dettaglio fasce A/B/C
6. Test responsive mobile (info panel diventa bottom sheet)
7. Verificare navbar homepage con dropdown "Mappe" funzionante
